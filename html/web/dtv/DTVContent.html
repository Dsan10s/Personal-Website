During my internship with DIRECTV, I worked on an independent project coding a proof of concept application for the set top box on the Interactive Television team. The project was a Formula 1 Racing application that gave F1 fans an easier way to view different kinds of racing information, all while still watching TV. I built the application from scratch, both on the front-end and back-end side, with help and advice coming from my mentor throughout the duration of the project.

  • Languages/Technologies used:
    • Front End:
      • HTML, SASS, JavaScript
      • Google Closure
    • Back End:
      • Java
      • MongoDB
  • Formula 1 Racing Application Proof of Concept
    • Race Schedule
      • Users can choose from the various Grand Prix, and chooseone to learn the most up to date information about it
    • Race Roster
      • Gives users a place to learn more information about the racers, including their height, weight, date of birth, what car they're driving, and their country of origin
    • Race Standings
      • Users can quickly see the results of a race
    • Track Information
      • Users can see an image of the track
      • Number of laps, circuit length, race distance
      • Users can tab through the turns on the track to view how quickly the racers generally take that specific turn
