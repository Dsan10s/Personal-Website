Fritter is a Twitter clone that I made as one of my projects for my Software Construction class.

The application has fully implemented front end and back end systems working together, so users are able to make accounts and interact with the site.

  • Languages/Technologies used:
    • Front End:
      • HTML, CSS, JavaScript, jQuery
      • Bootstrap
    • Back End:
      • Node.js
      • MongoDB
      • Used Mongoose and Express.js within Node.js
    • Front End to Back End Communication
      • Form submission
  • Features:
    • Creating and deleting accounts
    • CRUD "freets"
    • Following/Unfollowing other users
    • Replying to freets
